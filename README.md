# Web Notebook

Web Notebook adalah aplikasi catatan berbasis web dengan tampilan modern, bersih, dan mudah digunakan. Aplikasi ini dibuat menggunakan HTML, CSS, dan JavaScript murni serta menggunakan SweetAlert2 untuk notifikasi agar pengalaman pengguna lebih profesional.

---

## Tujuan Proyek
- Membuat aplikasi catatan berbasis web yang praktis dan ringan
- Melatih penggunaan HTML, CSS responsive, dan JavaScript DOM
- Menerapkan CRUD (Create, Read, Update, Delete) tanpa alert bawaan browser
- Menjadi project portfolio frontend developer

---

## Fitur Utama
- Tambah catatan
- Edit catatan langsung di card
- Hapus catatan dengan SweetAlert2 confirmation modal
- Notifikasi toast sukses dan error
- Pencarian catatan secara realtime
- Responsive di semua ukuran layar
- Data tersimpan otomatis menggunakan LocalStorage

---

## Teknologi yang Digunakan
- HTML5
- CSS3
- JavaScript
- SweetAlert2
